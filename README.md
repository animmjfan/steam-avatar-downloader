# steam-avatar-downloader
This was inspired from a non-working repo of the same name I found while searching online which seemed botted.

## How to use
1. Clone this repo using `git clone https://github.com/animmjfan/steam-avatar-downloader.git` then cd into it
2. Open `steam-avatar-downloader.sh` in a text editor
3. Change the string YOURPROFILEIDHERE to your profile id or custom url \(both are supported\) then save the file
4. Run `steam-avatar-downloader.sh`
5. Profit

> [!NOTE]
> The image is saved as `image.jpg`
